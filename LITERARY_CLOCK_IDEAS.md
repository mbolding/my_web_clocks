## General & Creative Concepts

### Implemented
- ✅ **Atari Asteroids** - A tribute to the classic arcade game vector graphics.
- ✅ **Sun & Moon** - Shows sky position based on browser location and time.
- ✅ **Random Pomodoro** - Focus timer with randomized work (5-45m) and break (5-15m) intervals.
- ✅ **The Mealtime Clock** - Tells time by mealtimes, tea time, and coffee time.
- ✅ **Conference Clock** - Speaker timer with schedule and Q&A visualization.
- ✅ **Oscilloscope Clock** - XY-mode simulation of an analog oscilloscope.
- ✅ **Classic Mac OS** - System 6 style interface with draggable windows.
- ✅ **Vintage Clock** - Real photo background with dynamic overlaid hands.

## Design Notes & Themes

### Common Elements to Consider
- Typography matching the era/style
- Color palettes reflecting mood
- Sound effects where appropriate
- Text animations (stream of consciousness, fragments, etc.)
- Visual motifs (butterflies, stars, clocks, etc.)
- Quote cycling
- Chapter/section navigation
- Character indicators
- Location displays
- Thematic backgrounds

### Technical Patterns
- Single HTML file, self-contained
- CSS custom properties for theming
- Responsive design (clamp for sizing)
- Web Audio API for sounds
- Canvas for complex graphics when needed
- Smooth animations with CSS
- Hour-to-content mapping
- Real-time updates
