# Literary Clock Ideas

A collection of literary works that would make amazing time-based visualizations.

## Currently Implemented

### Vancian Quartet
- ✅ **The Dying Sun** - Jack Vance's Dying Earth with exotic time periods
- ✅ **Spell Preparation Clock** - Vancian magic system with spell memorization
- ✅ **Cutz Chronometer** - Invented time units (Grand Chimes, Lesser Bells, Sestinas, Jiffies)
- ✅ **The Deodand Clock** - Supernatural entities governing each hour

- ✅ **Bloomsday** - James Joyce's Ulysses (18 episodes through June 16, 1904)

## To Build

### Kafka & Existential

- **The Trial** - Josef K's bureaucratic nightmare
  - Endless waiting rooms
  - Absurd legal proceedings mapped to hours
  - Mounting anxiety and confusion
  - "Someone must have slandered Josef K..."
  - Bureaucratic typography

- **The Castle** - K's futile attempts to reach the authorities
  - Endless deferral and inaccessibility
  - Village vs. Castle
  - Surveyor's impossible task

### Time-Bending & Non-Linear

- **The Sound and the Fury** - William Faulkner
  - Four sections, four narrators (Benjy, Quentin, Jason, Dilsey)
  - Benjy's stream of consciousness (no time sense)
  - Quentin's obsession with time and watches
  - Different typefaces for different consciousness levels
  - April 7, 1928 and June 2, 1910


- **Cloud Atlas** - David Mitchell
  - Six nested stories across time
  - 1850s Pacific → distant post-apocalyptic future
  - Each hour could represent different time period
  - Interconnected birthmark motif

### Epic & Classical


- **Orlando** - Virginia Woolf
  - Centuries passing in one life
  - Gender transformation
  - Elizabethan → 1920s
  - Each hour represents different century
  - Shifting pronouns and identity

## Design Notes & Themes

### Common Elements to Consider
- Typography matching the era/style
- Color palettes reflecting mood
- Sound effects where appropriate
- Text animations (stream of consciousness, fragments, etc.)
- Visual motifs (butterflies, stars, clocks, etc.)
- Quote cycling
- Chapter/section navigation
- Character indicators
- Location displays
- Thematic backgrounds

### Technical Patterns
- Single HTML file, self-contained
- CSS custom properties for theming
- Responsive design (clamp for sizing)
- Web Audio API for sounds
- Canvas for complex graphics when needed
- Smooth animations with CSS
- Hour-to-content mapping
- Real-time updates
